- ~.java <- 소스파일.
- ~.java 안의 내용물 <- 소스코드.
- 자바 바이트코드 : 자바 컴파일러에 의해 생성되는 코드.
                    이 컴파일러에 의해 생성되는 코드의 명령어 크기가 1바이트.
-
class FirstJavaProgram
{
    public static void main(String[] args)
    {
        System.out.println("Hello Java!");
    }
}

에서 FirstJavaProgram 이 '클래스'이름, main이 '메소드'이름.
- 중괄호{, }를 이용해서 '영역' 또는 '경계'를 형성.
따라서 클래스이름 이후의 중괄호는 클래스의 영역을 알리는 용도로 사용됐고,
그 안의 메소드이름에 이어 등장하는 중괄호는 메소드의 영역을 알리는 용도로 사용됨.

-명령문의 끝에는 ';'를 붙여 명령문의 끝을 표시해야함.

- 블록단위 주석 : /* ~ */
- 행 단위 주석 : //
